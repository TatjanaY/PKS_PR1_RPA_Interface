# 🧩 Вариант 34 — «Заявки ИТ отдела»

## 🔹 Задание 1. Перенос данных по заявкам ИТ

### Подготовка данных

С помощью **DeepSeek** сгенерируйте тестовые данные и создайте файл:

**IT.xlsx**
Лист: `Заявки`
Размер: **45 строк × 5 столбцов**

Структура таблицы:

| ID | Клиент | Проблема | Дата | Статус |

Заполните таблицу реалистичными значениями.

Создайте файл:

**Отчет_IT.xlsx**
Лист: `Импорт` — пустой

---

### Требуется

Создать проект в Puzzle RPA Studio и реализовать процесс, который:

1. Считывает данные с листа `Заявки`.
2. Полностью переносит таблицу в файл `Отчет_IT.xlsx`.
3. Выполняет вставку, начиная:

   * со строки 2
   * со столбца 0.
4. Заполняет ячейки:

   * A1 — «Отчет по заявкам ИТ отдела»
   * F1 — «Дата формирования»
   * F2 — текущую дату

---

## 🔹 Задание 2. Дополнение отчета по критическим заявкам

### Подготовка данных

В файле `IT.xlsx` с помощью **DeepSeek** создайте новый лист:

**Лист:** `Критические`
Размер: **10 строк × 4 столбца**

Структура таблицы:

| ID | Клиент | Проблема | Дата |

---

### Требуется

1. В Puzzle RPA Studio выполнить **дублирование проекта**, созданного в задании 1.
2. Работу выполнять только в продублированном проекте.
3. Доработать процесс так, чтобы робот дополнительно:

   * считывал данные с листа `Критические`,
   * записывал их в файл `Отчет_IT.xlsx`, начиная со строки 2 и столбца 5,
   * заполнял ячейки:

     * F1 — «Критические заявки»
     * J1 — «Всего записей»
     * J2 — 10
     * A56 — «Отчет сформирован автоматически»
     * A57 — фамилию и имя студента
