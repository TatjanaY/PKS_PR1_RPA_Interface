# 🧩 Вариант 5 — «Банк — платежи»

---

## 🔹 Задание 1. Перенос данных по платежам

### Подготовка данных

С помощью **DeepSeek** сгенерируйте тестовые данные и создайте файл:

**Платежи.xlsx**
Лист: `Платежи`
Размер: **80 строк × 5 столбцов**

Структура таблицы:

| ID транзакции | Клиент | Сумма | Дата | Статус |

Заполните таблицу реалистичными значениями.

Создайте файл:

**Отчет_платежи.xlsx**
Лист: `Импорт` — пустой

---

### Требуется

Создать проект в Puzzle RPA Studio и реализовать процесс, который:

1. Считывает данные с листа `Платежи`.
2. Полностью переносит таблицу в файл `Отчет_платежи.xlsx`.
3. Выполняет вставку, начиная:

   * со строки 2
   * со столбца 0.
4. Заполняет ячейки:

   * A1 — «Отчет по платежам»
   * F1 — «Дата формирования»
   * F2 — текущую дату

---

## 🔹 Задание 2. Дополнение отчета по проблемным платежам

### Подготовка данных

В файле `Платежи.xlsx` с помощью **DeepSeek** создайте новый лист:

**Лист:** `Ошибки`
Размер: **20 строк × 4 столбца**

Структура таблицы:

| ID транзакции | Клиент | Сумма | Причина ошибки |

---

### Требуется

1. В Puzzle RPA Studio выполнить **дублирование проекта**, созданного в задании 1.
2. Работу выполнять только в продублированном проекте.
3. Доработать процесс так, чтобы робот дополнительно:

   * считывал данные с листа `Ошибки`,
   * записывал их в файл `Отчет_платежи.xlsx`, начиная со строки 2 и столбца 6,
   * заполнял ячейки:

     * G1 — «Проблемные платежи»
     * K1 — «Всего записей»
     * K2 — 20
     * A103 — «Отчет сформирован автоматически»
     * A104 — фамилию и имя студента
