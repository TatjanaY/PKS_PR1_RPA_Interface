# 🧩 Вариант 32 — «Учёт студентов»

## 🔹 Задание 1. Перенос данных по студентам

### Подготовка данных

С помощью **DeepSeek** сгенерируйте тестовые данные и создайте файл:

**Студенты.xlsx**
Лист: `Учёба`
Размер: **40 строк × 4 столбца**

Структура таблицы:

| № | ФИО | Группа | Средний балл |

Заполните таблицу реалистичными значениями.

Создайте файл:

**Отчет_учёба.xlsx**
Лист: `Импорт` — пустой

---

### Требуется

Создать проект в Puzzle RPA Studio и реализовать процесс, который:

1. Считывает данные с листа `Учёба`.
2. Полностью переносит таблицу в файл `Отчет_учёба.xlsx`.
3. Выполняет вставку, начиная:

   * со строки 2
   * со столбца 0.
4. Заполняет ячейки:

   * A1 — «Отчет по учёту студентов»
   * E1 — «Дата формирования»
   * E2 — текущую дату

---

## 🔹 Задание 2. Дополнение отчета по новым студентам

### Подготовка данных

В файле `Студенты.xlsx` с помощью **DeepSeek** создайте новый лист:

**Лист:** `Новые`
Размер: **10 строк × 4 столбца**

Структура таблицы:

| № | ФИО | Группа | Средний балл |

---

### Требуется

1. В Puzzle RPA Studio выполнить **дублирование проекта**, созданного в задании 1.
2. Работу выполнять только в продублированном проекте.
3. Доработать процесс так, чтобы робот дополнительно:

   * считывал данные с листа `Новые`,
   * записывал их в файл `Отчет_учёба.xlsx`, начиная со строки 2 и столбца 5,
   * заполнял ячейки:

     * F1 — «Новые студенты»
     * J1 — «Всего записей»
     * J2 — 10
     * A51 — «Отчет сформирован автоматически»
     * A52 — фамилию и имя студента
