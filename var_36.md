# 🧩 Вариант 36 — «Учёт библиотечных журналов»

## 🔹 Задание 1. Перенос данных по журналам

### Подготовка данных

С помощью **DeepSeek** сгенерируйте тестовые данные и создайте файл:

**Журналы.xlsx**
Лист: `Учёт`
Размер: **30 строк × 4 столбца**

Структура таблицы:

| ID | Название | Год | Кол-во экземпляров |

Заполните таблицу реалистичными значениями.

Создайте файл:

**Отчет_журналы.xlsx**
Лист: `Импорт` — пустой

---

### Требуется

Создать проект в Puzzle RPA Studio и реализовать процесс, который:

1. Считывает данные с листа `Учёт`.
2. Полностью переносит таблицу в файл `Отчет_журналы.xlsx`.
3. Выполняет вставку, начиная:

   * со строки 2
   * со столбца 0.
4. Заполняет ячейки:

   * A1 — «Отчет по библиотечным журналам»
   * E1 — «Дата формирования»
   * E2 — текущую дату

---

## 🔹 Задание 2. Дополнение отчета по новым журналам

### Подготовка данных

В файле `Журналы.xlsx` с помощью **DeepSeek** создайте новый лист:

**Лист:** `Новые`
Размер: **10 строк × 4 столбца**

Структура таблицы:

| ID | Название | Год | Кол-во экземпляров |

---

### Требуется

1. В Puzzle RPA Studio выполнить **дублирование проекта**, созданного в задании 1.
2. Работу выполнять только в продублированном проекте.
3. Доработать процесс так, чтобы робот дополнительно:

   * считывал данные с листа `Новые`,
   * записывал их в файл `Отчет_журналы.xlsx`, начиная со строки 2 и столбца 5,
   * заполнял ячейки:

     * F1 — «Новые журналы»
     * J1 — «Всего записей»
     * J2 — 10
     * A41 — «Отчет сформирован автоматически»
     * A42 — фамилию и имя студента
