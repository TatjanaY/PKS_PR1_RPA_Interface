# 🧩 Вариант 8 — «Заявки техподдержки»

---

## 🔹 Задание 1. Перенос данных по заявкам

### Подготовка данных

С помощью **DeepSeek** сгенерируйте тестовые данные и создайте файл:

**Заявки.xlsx**
Лист: `Заявки`
Размер: **50 строк × 5 столбцов**

Структура таблицы:

| ID | Клиент | Проблема | Дата | Статус |

Заполните таблицу реалистичными значениями.

Создайте файл:

**Отчет_заявки.xlsx**
Лист: `Импорт` — пустой

---

### Требуется

Создать проект в Puzzle RPA Studio и реализовать процесс, который:

1. Считывает данные с листа `Заявки`.
2. Полностью переносит таблицу в файл `Отчет_заявки.xlsx`.
3. Выполняет вставку, начиная:

   * со строки 2
   * со столбца 0.
4. Заполняет ячейки:

   * A1 — «Отчет по заявкам техподдержки»
   * F1 — «Дата формирования»
   * F2 — текущую дату

---

## 🔹 Задание 2. Дополнение отчета обработанными заявками

### Подготовка данных

В файле `Заявки.xlsx` с помощью **DeepSeek** создайте новый лист:

**Лист:** `Обработано`
Размер: **30 строк × 4 столбца**

Структура таблицы:

| ID | Клиент | Дата обработки | Результат |

---

### Требуется

1. В Puzzle RPA Studio выполнить **дублирование проекта**, созданного в задании 1.
2. Работу выполнять только в продублированном проекте.
3. Доработать процесс так, чтобы робот дополнительно:

   * считывал данные с листа `Обработано`,
   * записывал их в файл `Отчет_заявки.xlsx`, начиная со строки 2 и столбца 6,
   * заполнял ячейки:

     * G1 — «Обработанные заявки»
     * K1 — «Всего записей»
     * K2 — 30
     * A83 — «Отчет сформирован автоматически»
     * A84 — фамилию и имя студента
