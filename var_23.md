# 🧩 Вариант 23 — «Медицинские анализы»

## 🔹 Задание 1. Перенос данных по анализам

### Подготовка данных

С помощью **DeepSeek** сгенерируйте тестовые данные и создайте файл:

**Анализы.xlsx**
Лист: `Результаты`
Размер: **40 строк × 5 столбцов**

Структура таблицы:

| ID | ФИО | Анализ | Дата | Результат |

Заполните таблицу реалистичными значениями.

Создайте файл:

**Отчет_анализы.xlsx**
Лист: `Импорт` — пустой

---

### Требуется

Создать проект в Puzzle RPA Studio и реализовать процесс, который:

1. Считывает данные с листа `Результаты`.
2. Полностью переносит таблицу в файл `Отчет_анализы.xlsx`.
3. Выполняет вставку, начиная:

   * со строки 2
   * со столбца 0.
4. Заполняет ячейки:

   * A1 — «Отчет по медицинским анализам»
   * F1 — «Дата формирования»
   * F2 — текущую дату

---

## 🔹 Задание 2. Дополнение отчета по повторным анализам

### Подготовка данных

В файле `Анализы.xlsx` с помощью **DeepSeek** создайте новый лист:

**Лист:** `Повторные`
Размер: **15 строк × 4 столбца**

Структура таблицы:

| ID | ФИО | Анализ | Дата повторного исследования |

---

### Требуется

1. В Puzzle RPA Studio выполнить **дублирование проекта**, созданного в задании 1.
2. Работу выполнять только в продублированном проекте.
3. Доработать процесс так, чтобы робот дополнительно:

   * считывал данные с листа `Повторные`,
   * записывал их в файл `Отчет_анализы.xlsx`, начиная со строки 2 и столбца 6,
   * заполнял ячейки:

     * G1 — «Повторные анализы»
     * K1 — «Всего записей»
     * K2 — 15
     * A57 — «Отчет сформирован автоматически»
     * A58 — фамилию и имя студента
