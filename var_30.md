# 🧩 Вариант 30 — «Финансовые транзакции»

## 🔹 Задание 1. Перенос данных по транзакциям

### Подготовка данных

С помощью **DeepSeek** сгенерируйте тестовые данные и создайте файл:

**Транзакции.xlsx**
Лист: `Операции`
Размер: **60 строк × 5 столбцов**

Структура таблицы:

| ID | Клиент | Сумма | Дата | Тип |

Заполните таблицу реалистичными значениями.

Создайте файл:

**Отчет_транзакции.xlsx**
Лист: `Импорт` — пустой

---

### Требуется

Создать проект в Puzzle RPA Studio и реализовать процесс, который:

1. Считывает данные с листа `Операции`.
2. Полностью переносит таблицу в файл `Отчет_транзакции.xlsx`.
3. Выполняет вставку, начиная:

   * со строки 2
   * со столбца 0.
4. Заполняет ячейки:

   * A1 — «Отчет по финансовым транзакциям»
   * F1 — «Дата формирования»
   * F2 — текущую дату

---

## 🔹 Задание 2. Дополнение отчета по возвратам транзакций

### Подготовка данных

В файле `Транзакции.xlsx` с помощью **DeepSeek** создайте новый лист:

**Лист:** `Возвраты`
Размер: **15 строк × 4 столбца**

Структура таблицы:

| ID | Клиент | Сумма | Дата |

---

### Требуется

1. В Puzzle RPA Studio выполнить **дублирование проекта**, созданного в задании 1.
2. Работу выполнять только в продублированном проекте.
3. Доработать процесс так, чтобы робот дополнительно:

   * считывал данные с листа `Возвраты`,
   * записывал их в файл `Отчет_транзакции.xlsx`, начиная со строки 2 и столбца 5,
   * заполнял ячейки:

     * F1 — «Возвраты»
     * J1 — «Всего записей»
     * J2 — 15
     * A76 — «Отчет сформирован автоматически»
     * A77 — фамилию и имя студента
