# 🧩 Вариант 28 — «Заказы кафе»

## 🔹 Задание 1. Перенос данных по заказам

### Подготовка данных

С помощью **DeepSeek** сгенерируйте тестовые данные и создайте файл:

**Кафе.xlsx**
Лист: `Заказы`
Размер: **50 строк × 5 столбцов**

Структура таблицы:

| № заказа | Блюдо | Кол-во | Сумма | Дата |

Заполните таблицу реалистичными значениями.

Создайте файл:

**Отчет_кафе.xlsx**
Лист: `Импорт` — пустой

---

### Требуется

Создать проект в Puzzle RPA Studio и реализовать процесс, который:

1. Считывает данные с листа `Заказы`.
2. Полностью переносит таблицу в файл `Отчет_кафе.xlsx`.
3. Выполняет вставку, начиная:

   * со строки 2
   * со столбца 0.
4. Заполняет ячейки:

   * A1 — «Отчет по заказам кафе»
   * F1 — «Дата формирования»
   * F2 — текущую дату

---

## 🔹 Задание 2. Дополнение отчета по акциям дня

### Подготовка данных

В файле `Кафе.xlsx` с помощью **DeepSeek** создайте новый лист:

**Лист:** `Акции`
Размер: **15 строк × 4 столбца**

Структура таблицы:

| Блюдо | Скидка % | Дата | Кол-во |

---

### Требуется

1. В Puzzle RPA Studio выполнить **дублирование проекта**, созданного в задании 1.
2. Работу выполнять только в продублированном проекте.
3. Доработать процесс так, чтобы робот дополнительно:

   * считывал данные с листа `Акции`,
   * записывал их в файл `Отчет_кафе.xlsx`, начиная со строки 2 и столбца 5,
   * заполнял ячейки:

     * F1 — «Акции дня»
     * J1 — «Всего записей»
     * J2 — 15
     * A67 — «Отчет сформирован автоматически»
     * A68 — фамилию и имя студента
