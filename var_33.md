# 🧩 Вариант 33 — «Магазин бытовой техники»

## 🔹 Задание 1. Перенос данных по продажам техники

### Подготовка данных

С помощью **DeepSeek** сгенерируйте тестовые данные и создайте файл:

**Техника.xlsx**
Лист: `Продажи`
Размер: **55 строк × 5 столбцов**

Структура таблицы:

| ID | Наименование | Категория | Кол-во | Цена |

Заполните таблицу реалистичными значениями.

Создайте файл:

**Отчет_техника.xlsx**
Лист: `Импорт` — пустой

---

### Требуется

Создать проект в Puzzle RPA Studio и реализовать процесс, который:

1. Считывает данные с листа `Продажи`.
2. Полностью переносит таблицу в файл `Отчет_техника.xlsx`.
3. Выполняет вставку, начиная:

   * со строки 2
   * со столбца 0.
4. Заполняет ячейки:

   * A1 — «Отчет по продажам бытовой техники»
   * F1 — «Дата формирования»
   * F2 — текущую дату

---

## 🔹 Задание 2. Дополнение отчета по гарантийным случаям

### Подготовка данных

В файле `Техника.xlsx` с помощью **DeepSeek** создайте новый лист:

**Лист:** `Гарантии`
Размер: **15 строк × 4 столбца**

Структура таблицы:

| ID | Наименование | Проблема | Дата обращения |

---

### Требуется

1. В Puzzle RPA Studio выполнить **дублирование проекта**, созданного в задании 1.
2. Работу выполнять только в продублированном проекте.
3. Доработать процесс так, чтобы робот дополнительно:

   * считывал данные с листа `Гарантии`,
   * записывал их в файл `Отчет_техника.xlsx`, начиная со строки 2 и столбца 5,
   * заполнял ячейки:

     * F1 — «Гарантийные случаи»
     * J1 — «Всего записей»
     * J2 — 15
     * A71 — «Отчет сформирован автоматически»
     * A72 — фамилию и имя студента
