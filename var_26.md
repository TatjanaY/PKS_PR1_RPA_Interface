# 🧩 Вариант 26 — «Продажи супермаркета»

## 🔹 Задание 1. Перенос данных по продажам

### Подготовка данных

С помощью **DeepSeek** сгенерируйте тестовые данные и создайте файл:

**Супермаркет.xlsx**
Лист: `Продажи`
Размер: **70 строк × 5 столбцов**

Структура таблицы:

| ID | Товар | Категория | Кол-во | Цена |

Заполните таблицу реалистичными значениями.

Создайте файл:

**Отчет_супермаркет.xlsx**
Лист: `Импорт` — пустой

---

### Требуется

Создать проект в Puzzle RPA Studio и реализовать процесс, который:

1. Считывает данные с листа `Продажи`.
2. Полностью переносит таблицу в файл `Отчет_супермаркет.xlsx`.
3. Выполняет вставку, начиная:

   * со строки 2
   * со столбца 0.
4. Заполняет ячейки:

   * A1 — «Отчет по продажам супермаркета»
   * F1 — «Дата формирования»
   * F2 — текущую дату

---

## 🔹 Задание 2. Дополнение отчета по акциям

### Подготовка данных

В файле `Супермаркет.xlsx` с помощью **DeepSeek** создайте новый лист:

**Лист:** `Акции`
Размер: **20 строк × 4 столбца**

Структура таблицы:

| ID | Товар | Скидка % | Дата начала |

---

### Требуется

1. В Puzzle RPA Studio выполнить **дублирование проекта**, созданного в задании 1.
2. Работу выполнять только в продублированном проекте.
3. Доработать процесс так, чтобы робот дополнительно:

   * считывал данные с листа `Акции`,
   * записывал их в файл `Отчет_супермаркет.xlsx`, начиная со строки 2 и столбца 6,
   * заполнял ячейки:

     * G1 — «Акции»
     * K1 — «Всего записей»
     * K2 — 20
     * A91 — «Отчет сформирован автоматически»
     * A92 — фамилию и имя студента
