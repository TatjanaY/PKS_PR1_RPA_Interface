# 🧩 Вариант 18 — «Магазин одежды»

## 🔹 Задание 1. Перенос данных по товарам

### Подготовка данных

С помощью **DeepSeek** сгенерируйте тестовые данные и создайте файл:

**Одежда.xlsx**
Лист: `Ассортимент`
Размер: **60 строк × 5 столбцов**

Структура таблицы:

| ID | Наименование | Категория | Размер | Цена |

Заполните таблицу реалистичными значениями.

Создайте файл:

**Отчет_одежда.xlsx**
Лист: `Импорт` — пустой

---

### Требуется

Создать проект в Puzzle RPA Studio и реализовать процесс, который:

1. Считывает данные с листа `Ассортимент`.
2. Полностью переносит таблицу в файл `Отчет_одежда.xlsx`.
3. Выполняет вставку, начиная:

   * со строки 2
   * со столбца 0.
4. Заполняет ячейки:

   * A1 — «Отчет по магазину одежды»
   * F1 — «Дата формирования»
   * F2 — текущую дату

---

## 🔹 Задание 2. Дополнение отчета по скидкам

### Подготовка данных

В файле `Одежда.xlsx` с помощью **DeepSeek** создайте новый лист:

**Лист:** `Скидки`
Размер: **20 строк × 4 столбца**

Структура таблицы:

| ID | Наименование | Скидка % | Дата начала |

---

### Требуется

1. В Puzzle RPA Studio выполнить **дублирование проекта**, созданного в задании 1.
2. Работу выполнять только в продублированном проекте.
3. Доработать процесс так, чтобы робот дополнительно:

   * считывал данные с листа `Скидки`,
   * записывал их в файл `Отчет_одежда.xlsx`, начиная со строки 2 и столбца 6,
   * заполнял ячейки:

     * G1 — «Скидки»
     * K1 — «Всего записей»
     * K2 — 20
     * A83 — «Отчет сформирован автоматически»
     * A84 — фамилию и имя студента
